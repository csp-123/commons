# 阴阳师脚本：

## 支持副本：
1. 御魂
2. 觉醒 
3. 探索
4. 自定义（活动定制）
## 支持挂机模式：
1. 单刷
2. 组队
## 支持挂机时间：
1. 限时
2. 限次
3. 无限刷
## 容错机制：
1. 好友弹出悬赏
2. 体力不足
3. buff开关
## 防封措施：
思路：尽可能模拟人类行为，使脚本的操作更加类似手动操作。
1. 鼠标不能点击太快 太频繁
2. 点击时避免固定一个位置
3. 避免点击释放频率相同
4. 间隔频率不能相同
5. 增加一些模拟行为
